# RandomizedEverything
A randomizer for Minecraft

## Tutorial
1) Get a Minecraft version from your Minecraft "versions" folder
2) Run the code (Or make a folder named "Version" in the folder with the .py
3) Copy the Minecraft version into the Version folder
4) Run the code again
5) Code should function and generate a datapack

### Tested and funtional with 1.20.4, not sure if this works with older/newer versions
